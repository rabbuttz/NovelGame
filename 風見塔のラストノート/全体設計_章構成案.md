## 第1章「転入初日、封鎖塔と仮チーム」
- **章の目的**
  - 主人公の転入導入、風見塔との遭遇、主要人物の第一印象提示。

- **主に掘るキャラ**
  - きりゅう / あむ / さわでん（導入トリオ）

- **入れるべきイベント**

  - 「想いを伝えると結ばれる塔」の噂初出

- **必須の感情変化**
  - 主人公: 「部外者」→「巻き込まれた当事者」
  - きりゅう: 警戒→最低限の信頼
  - あむ: 興味本位→仲間意識の芽
- **分岐に使う変数やフラグ候補**

  - `flags.acceptedErrandRole`
  - `flags.heardTowerRumor`
  - `affection.kiryu / affection.amu / affection.agasa / affection.sawaden`
- **章末の引き**
  - 風見塔内部の古い掲示に「最終公開記録」の文言。誰が最後に使ったのか不明。

### 第9章の具体案

- **章タイトル案**
  - 「止める大人、支える大人」
  - もしくは「先生たちの白線」

- **この章で必ず達成したいこと**
  - 第8章までの「生徒たちの感情回」を受けて、次は大人側にも譲れない理由があると示す。
  - ロクロを単なる障害役から、過去の責任を背負って線を引く人物へ格上げする。
  - ごんたを一時的な便利キャラではなく、「現場で支える教師」として本筋へ接続する。
  - きりゅうに「生徒側の責任者」として一段上の視点を持たせる。
  - 章末で、閉鎖当時の記録に現メンバーとつながる名前を出し、第10章の記録掘りへ渡す。

- **主役配置**
  - メイン: ロクロ / ごんた / きりゅう
  - 補助: アガサ / あむ / 主人公
  - 影で効かせる: みもち（記録整理の導線役）

- **舞台候補**
  - 朝: 廊下 or 教室・昼
  - 午後: 体育館 or 保健室
  - 夕方: 図書館
  - 夜前: kazamito-night-overlook または 廊下

- **想定フラグ候補**
  - `flags.chapter9Clear`
  - `flags.heardRokuroReason`
  - `flags.heardGontaReason`
  - `flags.acceptedAdultBurden`
  - `flags.sawClosedRecordName`
  - `flags.backedKiryuResponsibility`
  - `trust.teacherSide`
  - `lore.closedTowerRecord`

- **好感度の主な動き**
  - `affection.kiryu`: 責任を一緒に引き受ける選択で上昇
  - `affection.agasa`: 交渉より説明責任を重視する選択で微上昇
  - `affection.amu`: 大人への反発を頭ごなしに抑えず受け止める選択で微上昇

- **シーン構成案**

#### 1. 朝の差し戻し
- 内容
  - 前日の確認結果を出した直後、ロクロが追加書類を机に置く。
  - 内容は「公開導線」ではなく「事故時対応」「責任分界」「教員立会い計画」の追記要求。
  - 生徒側は「また増えた」と反発気味だが、ロクロはいつもより言葉を選ばない。
- ここで見せるもの
  - ロクロが厳しいのは感情ではなく、書けない責任を通せないから。
  - あむやうきわ系の「気持ちはあるのに書類で止まる」苛立ち。
- 選択肢案
  - 「なぜそこまで細かく必要なのか聞く」
  - 「まず不足分を埋めようと提案する」
- 反映
  - 前者で `flags.acceptedAdultBurden` の下地
  - 後者で `affection.kiryu` 微上昇

#### 2. 体育館裏の再配置
- 内容
  - ごんたが備品確認を手伝いながら、「事故は本番より帰り際に起きる」と現場目線で説明する。
  - 以前の番外編で見せた軽さを残しつつ、今回は教師としての真顔を出す。
  - 体力差、混雑、パニック時の動線など、生徒が見落としやすい視点を入れる。
- ここで見せるもの
  - ごんたの変な人らしさと、危機察知の鋭さの両立。
  - 「支える大人」は勢いを止めるのではなく、倒れない形に整えていること。
- 選択肢案
  - 「人の流れより、怪我人の搬送を優先して質問する」
  - 「見学者の安全確保を優先して質問する」
- 反映
  - `flags.heardGontaReason: true`
  - どちらを選んでも進むが、第12章の演出差分に使える

#### 3. きりゅうとの廊下会話
- 内容
  - きりゅうが「止める側の言い分はわかる。でもわかるだけで通す理由にはならない」とこぼす。
  - 主人公にだけ、「責任を負うって、たぶん好かれることじゃない」と本音を見せる。
  - ここで生徒側責任者としての孤独を掘る。
- ここで見せるもの
  - きりゅうが大人側と対立しているのではなく、同じ地平で考え始めていること。
  - 主人公が“手伝い役”から“一緒に背負う側”へ移る節目。
- 選択肢案
  - 「一緒に嫌われ役を引き受ける」
  - 「きりゅうが前に出るなら支えると言う」
- 反映
  - `flags.backedKiryuResponsibility: true`
  - 前者は `affection.kiryu` 大きめ上昇

#### 4. 図書館での記録照合
- 内容
  - みもちが古い貸出記録や校内報を持ってきて、閉鎖時期の資料にズレがあると指摘する。
  - ロクロも合流し、隠していたわけではなく「正式記録と口伝が噛み合っていない」と説明する。
  - 過去の「事故」と呼ばれているものが、実は複数の出来事を一括で語られている可能性が出る。
- ここで見せるもの
  - ミステリが個人の思い出話ではなく、記録の断絶として拡張されること。
  - みもちの資料整理能力を終盤でも効かせる。
- 選択肢案
  - 「欠けているのは意図的だと思う」
  - 「混乱の中で散逸した可能性を考える」
- 反映
  - `lore.closedTowerRecord += 1`
  - 第10章で拾う仮説分岐の種

#### 5. ロクロの告白
- 内容
  - ロクロが閉鎖当時、自分はまだ新任寄りで、最終判断の場にいたことを明かす。
  - 自分が止め切れなかったこと、逆に必要以上に一括停止して残した痛み、その両方を背負っていると話す。
  - 「僕は反対しているんじゃない。繰り返させない形を見ている」と、これまでの立場が本音として回収される。
- ここで見せるもの
  - ロクロの責任と後悔
  - 生徒側が初めて「止める理由」の重さを理解する瞬間
- 選択肢案
  - 「だからこそ今回も慎重なんですね、と受け止める」
  - 「それでも今の自分たちを見てほしいと返す」
- 反映
  - `flags.heardRokuroReason: true`
  - `trust.teacherSide` 加算

#### 6. 章末フック: 閉鎖記録の署名
- 内容
  - みもちが整理していた紙束の末尾から、閉鎖当時の補足記録を発見する。
  - そこには、現メンバーの家族や兄姉、あるいは今の教師陣とつながる姓がある。
  - 直接答えは出さず、「今の物語は当時の当事者の続きでもある」とだけ示して切る。
- 出すべき引き
  - 例: 「音葉」「どりの後輩」「ロクロの旧担当生徒」とは別ラインの、もっと中心に近い名前
  - もしくは、アガサやきりゅうが知っていた可能性のある姓
- 次章への接続
  - 第10章で「もう一冊の記録」探索へ進む

- **YAML化する時のノード案**
  - `morning_revision_request`
  - `gonta_safety_walkthrough`
  - `kiryu_hallway_talk`
  - `library_record_check`
  - `rokuro_confession`
  - `record_name_hook`
  - `ending`

- **この章の空気感メモ**
  - これまでより少し静かで、会話の重さを優先する。
  - 泣きや大爆発ではなく、説明責任と後悔の重みで読ませる章。
  - そのぶん、ごんたの少し変な台詞で空気を詰めすぎない。

- **やってはいけないこと**
  - ロクロを急に全面的な味方にしすぎない。
  - ごんたを便利な解決装置にしない。
  - 事故の詳細をここで全部明かさない。第10章以降へ必ず余白を残す。
  - 生徒側の熱意を「子どもの無謀」として片づけない。

## 第10章「もう一冊の記録と、塔を使った人たちの過去」
- **章の目的**
  - ラストノート周辺の謎をさらに掘り下げる。
  - 噂のロマンだけでなく、記録として残された痛みを見せる。
- **主に掘るキャラ**
  - みもち / さわでん / ぐら / ねむのみ
- **入れるべきイベント**
  - 追加の記録断片回収
  - 展示物と現物資料の照合
  - 誰が何を残し、誰が何を消したのかの輪郭提示
- **章末の引き**
  - 欠落ページだけでなく、意図的に外された関連記録の存在が見つかる。

## 第11章「公開直前、もう一度崩れる均衡」
- **章の目的**
  - 追加章を重ねた上で、改めて最大危機を作る。
  - 共通ルート終盤の圧を十分に高める。
- **主に掘るキャラ**
  - アガサ / すずどら / あむ / うきわ
- **入れるべきイベント**
  - 差し止め再燃
  - チーム内の再衝突
  - 主人公による再調整と優先順位の決断
- **章末の引き**
  - 公開可否の最終判断が、ようやく目前まで来る。

## 第12章「星潮祭、風見塔の再点灯」
- **章の目的**
  - 共通ルート決着と個別導入フックの同時達成。
  - 成功/綱渡り成功の2系統で個別へ接続可能にする。
- **主に掘るキャラ**
  - 全体群像 + ルート確定上位キャラ
- **入れるべきイベント**
  - 正午決裁→公開可否決定
  - 一日限定公開スタート
  - 来場者反応（噂の再燃と鎮静）
  - 夜、塔最上階で「ラストノート」を巡る最終対話
  - 主人公が特定キャラと二人になる導線
- **必須の感情変化**
  - 主人公: 企画達成感→個人的な恋の自覚
  - チーム: 企画仲間→それぞれの未来へ
- **分岐に使う変数やフラグ候補**
  - `flags.commonRouteCleared`
  - `route.target`
  - `ending.commonOutcome`（perfect / narrow）
- **章末の引き**
  - 「この夜を境に、もう“ただの仲間”には戻れない」で個別ルート開始。
